INPORTANT NOTE!!!!!!!!!!!!!!!!! PLEASE NOTE THAT THIS GAME REQUIRES THE MODULE "PYGAME" VER 2.6.1 OR ABOVE


FAILURE TO HAVE THIS INSTALLED WILL RESULT IN THE AUDIO BEING MISSING AND THE SCRIPT RETURNING ERRORS
YOU SHOULD INSTALL PYGAME VIA THE FOLLOWING STEPS:
	- OPEN TERMINAL IN YOUR WORKING DIRECTORY OR POWERSHELL, BUT ENSURE YOUR COPY OF THE GAME CAN ACCESS THIS
	- RUN "pip install pygame"
	-IF THIS DOES NOT WORK, RUN "pip3 install pygame --target . --no-user"
